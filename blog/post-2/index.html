Hypothesis Testing 假設檢定

By Yao-Chung 25 October 2020

StatQuest (Hypothesis Testing) 假設檢定

筆記重點：

Null Hypothesis: There is no difference between things (虛無假設 代表兩者沒有差異)

We use hypothesis testing to determine we fail to reject the null hypothesis or to reject null hypothesis 我們做統計檢定來決定是否對虛無假設拒絕失敗或是拒絕虛無假設。

We reject the Null Hypothesis in favor of this Alternative Hypothesis. 這種寫法比較好，原因是因為我們不知道我們選擇的 Alternatice Hypothesis 是否為最佳的，所以用 in favor of 而不是說 accept。這部最後有整理說一個統計檢定需包含三個部分：(1) It needs data (2) Null Hypothesis (3) Alternative Hypothesis。當只有兩個族群的資料的時候，Alternative Hypothesis 非常的簡單就是 Null Hypothesis 的相反 (Mean1 =/= Mean2)。

統計 P 值（ p-value）是一個介於 0 至 1 之間的數字，量化我們對於 group A 和 group B (比如藥物 A 和藥物 B 對於疾病的療效） 有不同這件事情有多少的信心，當 p-value 越接近 0 的時候，我們對於差異的信心越大。常用的閾值是 0.05，意思就是當量夠大的時候，我們判斷錯誤的機率是 5% 。

Getting a small value of p-value when it is no difference is called False Positive. 制訂的閾值大小代表說你願意接受多少機率會失敗。以下為一個統計敘述句範例可以學習：If the p-value is 0.24, we are not confident that Drug A is different from Drug B. 最後要提醒的是，p-value 只能告訴我們這兩者有沒有顯著的差異，但他無法告速我們這樣的差異有多大。
